# ⬡ Flint

A **local-first**, secure knowledge base — a faithful clone of Obsidian, rebuilt from the ground up.

Your thoughts. Your data. Your control.

## Features

- **Markdown Editor** — Full GFM support with live preview, split view
- **Wiki Links** — `[[note name]]` syntax to connect ideas
- **Graph View** — Classic, minimal force-directed graph visualization
- **Local & Secure** — All data stored in your browser's localStorage. No cloud. No tracking.
- **Multi-vault** — Create separate workspaces for different projects
- **File Explorer** — Organize notes in folders with pinning support
- **Full-text Search** — Instant search across all notes
- **Backlinks** — See incoming and outgoing links for every note
- **Auto-save** — Never lose your work
- **Keyboard Shortcuts** — Power-user friendly

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New note |
| `Ctrl+E` | Cycle view modes (Edit → Preview → Split) |
| `Ctrl+G` | Toggle graph view |
| `Ctrl+Shift+F` | Search notes |
| `Ctrl+\` | Toggle sidebar |
| `Ctrl+S` | Save (auto-saved anyway) |

## Install

```bash
bash install.sh
```

## Update

```bash
bash update.sh
```

## Tech Stack

- React 19 + TypeScript
- Vite (single-file build)
- Tailwind CSS 4
- Canvas-based graph view

## Philosophy

Flint is built on the principle that your knowledge should be yours. No accounts, no cloud sync, no tracking. Everything stays on your machine, encrypted in your browser's local storage. Your vault persists across sessions.
